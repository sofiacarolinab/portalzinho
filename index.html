Como este portal foi feito

Este portal foi adaptado a partir das atividades do PDF; escolhi usar as mesmas tags, mas de forma que encaixasse no meu tema, além disso aproveitei para testar um pouco de CSS básico.

O que cada página pratica
  • Estrutura & menu: página inicial com título, parágrafo e links de navegação.
  • Galeria (galeria.html): imagens com <figure> e <figcaption>.
  • Tabelas (jogos.html): lista de jogos, notas e comentários em <table>.
  • Formulário (form.html): campos de texto, e-mail, número, radio e checkbox.
  • Âncoras (pet.html): navegação interna por seções com id.
  • Semântica: uso de <header> e <footer> nas páginas.
  • Interatividade sem JS: curiosidades de tubarões com <summary> + <details> (em curiosidades.html).
  • Observação: o mini-projeto de evento do PDF foi substituído/adaptado por este portal unificado.
