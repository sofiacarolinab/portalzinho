conteúdo explorado nessa página:
  • svg (sprite): bloco oculto com símbolos reutilizáveis (icon-estrela, icon-meia-estrela).
  • use: referencia cada símbolo do sprite para desenhar as estrelas.
  • div #jogos: contêiner da seção de avaliações.
  • h1: “jogos & opiniões”.
  • h4: “notas de 0 a 5 estrelas + comentário pessoal”.
  • br: quebra de linha entre título e tabela.
  • table .tabela: tabela das avaliações.
  • thead / tr / th: cabeçalho das colunas (jogo, nota, comentário).
  • tbody / tr / td: linhas com cada jogo.
  • td.jogo: nome do jogo.
  • td.nota: nota com ícones de estrela (SVG) ou texto.
  • td.comentario: comentário curto.
  • svg.estrela: ícone visual usado dentro da célula de nota.
